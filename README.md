# 🚀 Barkley Soluções Digitais

Landing page profissional desenvolvida com **Next.js (App Router)** simulando o site institucional de uma agência digital moderna focada em estratégia, tecnologia e crescimento de negócios.

Este projeto foi criado para aplicar conceitos avançados de **React + Next.js**, com arquitetura organizada, componentização e foco em performance.

---

## 🧠 Sobre o Projeto

A **Barkley Soluções Digitais** é uma agência fictícia especializada em:

* Web Design
* Marketing Digital
* SEO
* Consultoria Estratégica

O projeto simula um site real de apresentação de serviços com foco em:

* Conversão
* Autoridade
* Experiência do usuário
* Escalabilidade

---

## 🛠️ Tecnologias Utilizadas

* **Next.js (App Router)**
* **React**
* **CSS Modules**
* **Flexbox & Grid**
* **next/image**
* **Renderização Estática (SSG)**
* **TypeScript**

---


A estrutura segue o padrão do **App Router**, organizando a aplicação por responsabilidade e facilitando escalabilidade.

## 🚀 Como Rodar o Projeto

```bash
# Clonar repositório
git clone https://github.com/seu-usuario/barkley-solucoes-digitais

# Entrar na pasta
cd barkley-solucoes-digitais

# Instalar dependências
npm install

# Rodar projeto
npm run dev
```

Acesse:

```
http://localhost:3000
```
